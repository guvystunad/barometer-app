# Barometer & Altimeter

A simple, aviation-inspired Android app for measuring atmospheric pressure and altitude using phone's barometer sensor.

## Features

- Pressure reading in inHg
- Altitude calculation using barometric formula

## Requirements

- Android 7.0 (API 24) or higher
- Device with barometer sensor (pressure sensor)
- Location permission
